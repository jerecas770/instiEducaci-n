Materia 1: HTML

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje que se utiliza para estructurar una página web. Se escribe mediante etiquetas que indican al navegador cómo mostrar el contenido.

Ejemplo básico de código HTML:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi página web</title>
</head>
<body>
    <h1>Hola Mundo</h1>
    <p>Este es un párrafo de ejemplo.</p>
</body>
</html>

Etiquetas Comunes

  • Encabezados: <h1> a <h6>
  • Párrafos: <p>
  • Enlaces: <a href="">Texto</a>
  • Imágenes: <img src="" alt="">
  • Listas: <ul>, <ol>, <li>
  • Comentarios: <!-- Comentario -->

Materia 2: CSS

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje que permite dar estilo a los elementos HTML, como colores, tamaños, posiciones, etc.

Formas de usar CSS

  • En línea: style="color:red;"
  • Interno: dentro de <style> en el <head>
  • Externo: enlazado desde un archivo .css

Ejemplo de CSS Externo

/* estilos.css */
body {
    background-color: lightblue;
}
h1 {
    color: blue;
}
p {
    color: purple;
}

Selectores CSS

  • Etiqueta: p { color: red; }
  • ID: #parrafo1 { color: green; }
  • Clase: .centrado { text-align: center; }
  • Universal: * { font-family: Arial; }
