Materia 1: HTML

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje que se utiliza para estructurar una página web. Se escribe mediante etiquetas que indican al navegador cómo mostrar el contenido.

Ejemplo básico de código HTML:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi página web</title>
</head>
<body>
    <h1>Hola Mundo</h1>
    <p>Este es un párrafo de ejemplo.</p>
</body>
</html>

Etiquetas Comunes

  • Encabezados: <h1> a <h6>
  • Párrafos: <p>
  • Enlaces: <a href="">Texto</a>
  • Imágenes: <img src="" alt="">
  • Listas: <ul>, <ol>, <li>
  • Comentarios: <!-- Comentario -->
