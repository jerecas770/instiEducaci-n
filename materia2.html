HTML

En esta clase se vio HTML y sus conceptos básicos. Se explicó para qué sirve, cómo se usa para crear páginas web y se vieron ejemplos simples de títulos, párrafos, enlaces e imágenes. También se practicó con listas y tablas, usando editores como Visual Studio Code.

CSS

CSS significa Hojas de Estilo en Cascada y se utiliza para darle diseño y estilo a las páginas web. Con CSS podemos cambiar colores, tamaños de texto, márgenes, bordes y la disposición de los elementos. Es una herramienta fundamental para lograr que una página se vea atractiva y profesional.

tablas en HTML

Las tablas en HTML se utilizan para organizar información en filas y columnas. Permiten mostrar datos de manera clara y ordenada, como por ejemplo listas de precios, horarios o registros. Una tabla se compone de filas y dentro de cada fila se colocan las celdas que contienen los datos. También se pueden agregar títulos o encabezados para identificar cada columna. Las tablas se pueden personalizar con colores, bordes y diferentes estilos para mejorar su presentación.

¿Qué son las bases de datos y para qué sirven?

Las bases de datos son sistemas que permiten almacenar, organizar y acceder a grandes cantidades de información de forma rápida y segura. Están compuestas por colecciones de datos estructurados, normalmente organizados en tablas con filas y columnas. Se utilizan en casi todos los ámbitos: empresas, bancos, hospitales, gobiernos, redes sociales, y más. Gracias a ellas, es posible guardar datos de clientes, productos, empleados, transacciones y muchos otros tipos de información de manera ordenada y accesible. El principal objetivo de una base de datos es facilitar el acceso eficiente a la información, mantener su integridad y permitir su análisis. Además, ofrecen mecanismos de seguridad para proteger los datos sensibles y evitan errores o duplicaciones. Existen varios tipos, como las bases de datos relacionales (las más comunes), las no relacionales (para datos más flexibles), y las que se alojan en la nube o están distribuidas en varios servidores. En resumen, las bases de datos son fundamentales para el funcionamiento de casi cualquier sistema moderno que trabaje con datos.
